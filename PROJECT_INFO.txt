  ✅ 优雅的动画效果
  ✅ 响应式设计

────────────────────────────────────────────────────────────

  字体:     Outfit (现代几何字体)
  特效:     毛玻璃模糊 + 彩虹边框

📦 构建 APK 所需环境
────────────────────────────────────────────────────────────
  必需软件:
    ✓ Node.js (v14+)
    ✓ Java JDK (8 或 11)
    ✓ Android Studio
    ✓ Android SDK (API 22, 33)
    ✓ Gradle
  
  环境变量:
    ANDROID_HOME = Android SDK 路径
    PATH 包含 platform-tools

🚀 快速构建步骤
────────────────────────────────────────────────────────────
  1. 确保已安装 Android 开发环境
  2. 运行: ./cordova-build.sh
  3. 等待构建完成
  4. APK 位置: platforms/android/app/build/outputs/apk/debug/

📲 APK 输出
────────────────────────────────────────────────────────────
  调试版本:
    路径: platforms/android/app/build/outputs/apk/debug/app-debug.apk
    大小: ~3-5 MB
    特点: 可直接安装，包含调试符号
  
  发布版本:
    路径: platforms/android/app/build/outputs/apk/release/app-release-unsigned.apk
    大小: ~2-4 MB (优化后)
    特点: 需要签名才能安装

🛠️ 自动化构建脚本
────────────────────────────────────────────────────────────
  cordova-build.sh 功能:
    ✓ 自动检查所有依赖
    ✓ 智能环境提示
    ✓ 彩色进度输出
    ✓ 构建耗时统计
    ✓ APK 信息报告
    ✓ 下一步建议
  
  使用方法:
    ./cordova-build.sh          # 调试版本
    ./cordova-build.sh release  # 发布版本
    ./cordova-build.sh clean    # 清理重建
    ./cordova-build.sh run      # 构建并安装
    ./cordova-build.sh help     # 查看帮助
