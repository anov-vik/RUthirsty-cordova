# 应用界面说明

## 主界面布局

```
┌─────────────────────────────┐
│      💧 喝水打卡             │  ← 应用标题
│                             │
│  今日已喝水: 0 次            │  ← 今日统计(半透明白色背景)
│                             │
├─────────────────────────────┤
│                             │
│         ┌─────┐             │
│         │ 💧 │             │  ← 圆形打卡按钮(蓝色渐变)
│         │打卡 │             │     点击时有缩放动画
│         └─────┘             │
│                             │
├─────────────────────────────┤
│     喝水记录                │  ← 记录区域标题
│                             │
│  ┌─────────────────────┐   │
│  │ 2026/1/31 16:00  💧│   │  ← 记录卡片(紫色渐变)
│  └─────────────────────┘   │     新记录有滑入动画
│                             │
│  ┌─────────────────────┐   │
│  │ 2026/1/31 14:30  💧│   │
│  └─────────────────────┘   │
│                             │
│  (可滚动查看更多...)        │
│                             │
└─────────────────────────────┘
```

## 交互流程

1. **初次打开**
   - 显示"今日已喝水: 0 次"
   - 记录列表显示"暂无记录,点击上方按钮开始打卡吧!"

2. **点击打卡按钮**
   - 按钮播放缩放动画(0.5秒)
   - 今日次数 +1
   - 记录列表顶部添加新卡片
   - 新卡片播放滑入动画

3. **查看历史**
   - 向下滚动记录列表
   - 每条记录显示日期和时间
   - 最新记录在最上方

## 颜色方案

- **主背景**: 紫色到蓝色渐变 (#667eea → #764ba2)
- **统计区域**: rgba(255, 255, 255, 0.2) 半透明白色
- **打卡按钮**: 蓝色渐变 (#4facfe → #00f2fe)
- **记录卡片**: 紫色渐变 (#667eea → #764ba2)
- **文字颜色**: 白色
- **强调数字**: 金黄色 (#ffd700)

## 动画效果

### 打卡成功动画
```
按钮缩放: 1.0 → 1.1 → 1.0 (0.5秒)
```

### 新记录滑入动画
```
透明度: 0 → 1
位移: -10px → 0 (0.3秒)
```

## 响应式适配

- 自动适配各种屏幕尺寸
- 使用相对单位(px, %)
- 支持安全区域(刘海屏)
- 支持深色模式

## 数据显示格式

- **日期**: 2026/1/31 (本地化格式)
- **时间**: 16:00 (24小时制)
- **统计**: 今日已喝水: X 次

## 空状态

当没有记录时:
```
┌─────────────────────────────┐
│     喝水记录                │

│   暂无记录,点击上方按钮      │
│      开始打卡吧!            │
│                             │
└─────────────────────────────┘
```
